ISR ( USART1_UDRE_vect  )

This interrupt service routine will transmit all characters currently in the transmit buffer. Once all characters have been transmitted the interrupt will be disabled.

ISR ( USART1_RX_vect  )

This interrupt service routine will load every byte of data received via UART into the data buffer. Once the buffer is filled data received will no longer be loaded into the buffer.
